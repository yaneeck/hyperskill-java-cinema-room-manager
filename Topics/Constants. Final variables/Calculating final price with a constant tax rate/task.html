Calculating final price with a constant tax rate

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You are given a base price for a product represented as a floating point number. There's a constant tax rate of 10% added to the base price to compute the final price. Given the base price, write a Java program that calculates and prints the final price of the product. As input, your program should scan a floating point number that represents the base price and output the calculated final price rounded to two decimal places.
Sample Input:
10.00

Sample Output:
11.00


Sample Input:
20.50

Sample Output:
22.55



Memory limit: 256 MB
Time limit: 5 seconds
